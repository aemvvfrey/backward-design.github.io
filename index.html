Training to Run 13.1 Miles

Learning Objectives

This plan can be used to train to run a 5K, a 10K, a half marathon, or all three in that order.

  1. Train to run a 5K.
  2. Train to run a 10K.
  3. Train to run a Half Marathon.

Remember - Everybody's body is different, so please listen to yours. If you are feeling pain, (not just discomfort that happens to all of us from time to time when running), it may be that you are increasing your mileage too quickly or that you need to do more supplemental strength training.
