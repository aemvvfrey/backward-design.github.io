Learning Objective:

Train to Run a 5K

Follow the steps below to run a 5K in 6 weeks:

Week 1:

  • On day 1, run 1, then walk 2 minute. Repeat 10 times.
  • On day 2, run 2 min, then walk 4 minutes. Repeat 5 times.
  • On day 3, .run 2 min, then walk 4 minutes. Repeat 5 times.

Week 2:

  • On day 1, run 3 min, then walk 3 minutes. Repeat 4 times.
  • On day 2, run 3 min, then walk 3 minutes. Repeat 4 times.
  • On day 3, run 5 min, then walk 3 minutes. Repeat 3 times.

Week 3:

  • On day 1, run 7 min, then walk 2 minutes. Repeat 3 times.
  • On day 2, run 8 min, then walk 2 minutes. Repeat 3 times.
  • On day 3, run 8 min, then walk 2 minutes. Repeat 3 times..

Week 4:

  • On day 1, run 8 min, walk 2 minutes. Repeat 3 times.
  • On day 2, run 10 min, walk 12 minutes. Repeat 2 times then run 5 minutes.
  • On day 3, run 8 min, walk 2 minutes. Repeat 3 times.

Week 5:

  • On day 1, run 9 min, walk 1 minute. Repeat 3 times.
  • On day 2, run 12 min, walk 2 minutes. Repeat 2 times then run 5 minutes.
  • On day 3, run 8 min, walk 2 minutes. Repeat 3 times.

Week 6:

  • On day 1, run 15 min, walk 1 minute. Repeat 2 times.
  • On day 2, run 8 min, walk 2 minutes. Repeat 2 times. Repeat 3 times.
  • On day 3, run 3.1 miles - 5K!

Go back home.
